# RID122755_Desafio02

Desafio da Escola DNC — Desenvolvimento de um Blog Responsivo.

---

## Descrição do Projeto

Este projeto foi desenvolvido como parte do **Desafio: Desenvolvendo um Blog Responsivo**, proposto pela Escola DNC.  
O objetivo é criar um blog **totalmente responsivo**, garantindo uma experiência consistente e funcional em diferentes dispositivos, como smartphones, tablets, desktops e monitores ultrawide.

O desenvolvimento prioriza boas práticas de **HTML5 semântico**, **CSS responsivo**, uso de **Flexbox e Grid**, além da aplicação de **media queries** para adaptação do layout em múltiplas resoluções de tela.

---

## Tecnologias Utilizadas

- **HTML5** — Estrutura semântica do conteúdo.
- **CSS3** — Estilização responsiva utilizando Grid, Flexbox e media queries.
- **Google Fonts** — Fonte **Montserrat**, conforme especificação do protótipo.

> Não foi utilizado JavaScript, pois não é um requisito obrigatório para este desafio.
